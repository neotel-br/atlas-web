## Análise de Causa Raiz (RCA)

### O que aconteceu

O deploy da versão `v2.4.1` foi realizado sem a variável `OAUTH_CLIENT_SECRET` no ambiente de produção. O processo de CI/CD não possui validação de variáveis obrigatórias pré-deploy.

### Por que aconteceu

<Steps>
  <Step>
    A variável foi adicionada ao `.env.example` mas não ao Vault de secrets de produção.
  </Step>
  <Step>
    O pipeline de CD não executa healthcheck antes de remover o container anterior.
  </Step>
  <Step>
    O on-call primário estava em reunião — delay de 7 minutos no primeiro response.
  </Step>
</Steps>

### Por que não foi detectado antes

```
Deploy pipeline:
  ✅ Lint
  ✅ Unit tests
  ✅ Build
  ❌ Env vars validation  ← ausente
  ✅ Push imagem
  ✅ Deploy (sem rollback automático)
```
