## Autenticação JWT

A API usa **JWT (JSON Web Tokens)** para autenticação. Tokens de acesso expiram em **15 minutos**; use o refresh token para renová-los.

---

## Fluxo de Autenticação

<Steps>
  <Step>
    **Login** — envie credenciais e receba os tokens:

    ```http
    POST /v1/auth/login
    Content-Type: application/json

    {
      "email": "usuario@neotel.com.br",
      "password": "sua-senha"
    }
    ```

    Resposta:
    ```json
    {
      "access_token": "eyJhbGciOiJIUzI1NiIs...",
      "refresh_token": "eyJhbGciOiJIUzI1NiIs...",
      "expires_in": 900,
      "token_type": "Bearer"
    }
    ```
  </Step>
  <Step>
    **Use o access token** no header de todas as requisições:

    ```http
    GET /v1/docs
    Authorization: Bearer eyJhbGciOiJIUzI1NiIs...
    ```
  </Step>
  <Step>
    **Renove o token** quando expirar (HTTP 401):

    ```http
    POST /v1/auth/refresh
    Content-Type: application/json

    {
      "refresh_token": "eyJhbGciOiJIUzI1NiIs..."
    }
    ```
  </Step>
  <Step>
    **Logout** — invalide os tokens:

    ```http
    POST /v1/auth/logout
    Authorization: Bearer eyJhbGciOiJIUzI1NiIs...
    ```
  </Step>
</Steps>

---

## Erros de Autenticação

| Código | Mensagem | Causa |
|--------|---------|-------|
| `401` | `Unauthorized` | Token ausente ou inválido |
| `401` | `Token expired` | Access token expirado — faça refresh |
| `401` | `Refresh token expired` | Sessão expirada — faça login novamente |
| `403` | `Forbidden` | Token válido, mas sem permissão para o recurso |
| `429` | `Too Many Requests` | Rate limit excedido (100 req/min) |

<Callout type="warn">
  Refresh tokens expiram em **7 dias** de inatividade. Após isso, o usuário deve fazer login novamente.
</Callout>

## Exemplos por Linguagem

<Tabs items={['TypeScript', 'Python', 'cURL']}>
  <Tab value="TypeScript">
    ```typescript
    const response = await fetch('https://api.neotel.internal/v1/auth/login', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        email: 'usuario@neotel.com.br',
        password: 'senha',
      }),
    });

    const { access_token, refresh_token } = await response.json();

    // Requisição autenticada
    const docs = await fetch('https://api.neotel.internal/v1/docs', {
      headers: { Authorization: `Bearer ${access_token}` },
    });
    ```
  </Tab>
  <Tab value="Python">
    ```python
    import httpx

    client = httpx.Client(base_url="https://api.neotel.internal/v1")

    # Login
    resp = client.post("/auth/login", json={
        "email": "usuario@neotel.com.br",
        "password": "senha",
    })
    tokens = resp.json()

    # Requisição autenticada
    docs = client.get("/docs", headers={
        "Authorization": f"Bearer {tokens['access_token']}"
    })
    ```
  </Tab>
  <Tab value="cURL">
    ```bash
    # Login
    TOKEN=$(curl -s -X POST https://api.neotel.internal/v1/auth/login \
      -H "Content-Type: application/json" \
      -d '{"email":"usuario@neotel.com.br","password":"senha"}' \
      | jq -r '.access_token')

    # Requisição autenticada
    curl -H "Authorization: Bearer $TOKEN" \
      https://api.neotel.internal/v1/docs
    ```
  </Tab>
</Tabs>
